C++实现王浩算法
======
# 思路
先将输入的中缀表达式转换为后缀表达式，然后通过递归的方式实现相继式的化简，从而实现对命题的自动证明
# 程序说明
## 符号
"&"表示合取\n
"|"表示析取\n
"!"表示否定\n
">"表示蕴含\n
"="表示双蕴含\n
优先级:"()" > "!" > "&" > "|" > ">" > "="
## 输入
命题变项用大写字母表示，如A,B,C等\n
输入一个完整的中缀表达式，如：(!Q&(P>Q))>P
## 输出
若输入表达式为真，输出True，若为假则输出False\n
若存在上文规定外的符号，如#，小写字母等，输出input error
